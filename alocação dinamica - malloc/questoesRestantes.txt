Matriz Dinâmica: 
  Escreva um programa que peça ao usuário o número de linhas e colunas de uma matriz. Em seguida, aloque dinamicamente a memória para essa matriz (usando ponteiros duplos) e permita que o usuário insira valores em cada posição. Imprima a matriz e libere a memória.

Ordenação de Array Dinâmico: 
  Peça ao usuário para inserir n números e armazene-os em um array alocado dinamicamente. Ordene os números em ordem crescente e exiba o resultado. Não esqueça de liberar a memória no final.

Copiando Strings Dinamicamente: C
  rie uma função que receba uma string como parâmetro e use malloc para alocar uma nova string que seja uma cópia exata da original. A função deve retornar um ponteiro para a nova string. No main, imprima a string copiada e depois libere a memória.

Concatenação de Strings: 
  Escreva uma função que receba duas strings e retorne uma nova string que seja a concatenação das duas. Use malloc para alocar a quantidade exata de memória necessária para a string concatenada e, ao final, libere a memória.

Histograma Dinâmico de Caracteres: 
  Crie um programa que leia uma string do usuário e conte a frequência de cada caractere. Aloque dinamicamente um array que armazene essas frequências e imprima o resultado. Libere a memória ao final.

Divisão de Array Dinâmico: 
  Crie um programa que aloque dinamicamente um array de inteiros de tamanho n e o preencha com valores fornecidos pelo usuário. Em seguida, divida o array em dois novos arrays: um contendo os valores pares e outro os ímpares. Exiba os valores e libere a memória alocada.

Matriz Transposta Dinâmica: 
  Escreva um programa que peça ao usuário o número de linhas e colunas de uma matriz e, em seguida, aloque dinamicamente a matriz e permita que o usuário insira os valores. Calcule e imprima a matriz transposta e libere a memória.
